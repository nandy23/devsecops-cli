# DevSec

## The Open Source DevSecOps Assistant

## Vision

DevSec is an open source, enterprise-ready DevSecOps orchestration platform built for modern Platform Engineering teams.

Unlike traditional security scanners or pipeline generators, DevSec provides a unified way to:

* Analyze repositories
* Audit DevSecOps maturity
* Connect with enterprise DevSecOps platforms
* Generate secure CI/CD pipelines
* Recommend security best practices
* Produce unified security reports

The goal is to become the **kubectl** or **terraform** of DevSecOps.

---

# Mission

Enable every development team to adopt DevSecOps best practices with minimal effort.

DevSec should help organizations answer questions like:

* Are our repositories secure?
* Are we missing important security controls?
* Are our CI/CD pipelines following best practices?
* Which scanners should we use?
* How mature is our DevSecOps implementation?
* What should we improve next?
